# M5 Font Render

TTF font render support for M5Stack / M5Core2 with Arduino IDE.

![image]()


## How to use
1. Clone this repository into Arduino library directory.
2. Include ```M5FontRender.h``` **AFTER include MtStack.h / M5Core2.h**.
